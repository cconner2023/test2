<project_specification>
  <project_name>Beacon</project_name>

  <overview>
    Beacon is a military medical field application built for combat medics and medical teams. It provides triage algorithms from MEDCOM PAM 40-7-21, training tracking based on the 68W STP/CTL, E2EE messaging with LoRa mesh capability, custom text expanders for clinical note documentation, role-based auth across clinic hierarchies, and offline-first data persistence. This spec covers a hardening pass — fixing functionality, verifying security, adding test coverage, and ensuring offline-first reliability across all existing features. No new features, no UI changes.
  </overview>
